# Compare between IP address and MAC address in terms of:
- Structure.
- Usage.
- Location in the network layers.


|    | IP Address| MAC Address|
|----|----|-----|
|Stracture|It consists of numbers in the decimal system separated by periods and is 32 bits if it is IPv4 or 128 bits if it is IPv6|It consists of numbers in the hexadecimal system separated by a colon and is 48 bits long.|
|Usage|It is used to locate the device on the World Wide Web and transfer data between different networks.|It is used to uniquely locate the device within the local network for data transfer between directly connected devices.|
|  Location in the network layers  |Work in Network Layer|Work in Data Link Layer|
